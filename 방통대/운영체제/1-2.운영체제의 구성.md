운영체제

- 프로세스 관리자
- 메모리 관리자
- 파일 관리자
- 장치 관리자

프로세스 관리자

- 프로세스를 생성, 삭제, CPU 할당을 위한 스케줄 결정

프로세스란 : 실행되는 프로그램

메모리 관리자

- 메모리(주기억장치) 공간에 대한 요구의 유효성 체크
- 메모리 할당 및 회수
- 메모리 공간 보호

장치 관리자

- 컴퓨터 시스템의 모든 장치를 관리
- 시스템의 장치를 할당, 작동시작, 반환
- ex) 비디오 디스플레이, 키보드, 프린터 등등

파일 관리자

- 컴퓨터 시스템의 모든 파일을 관리
- 파일의 접근 제한 관리
- 파일을 열어 자원을 할당하거나 파일을 닫아 자원을 회수
